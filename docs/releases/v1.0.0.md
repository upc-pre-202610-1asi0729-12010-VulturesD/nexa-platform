# Nexa Platform v1.0.0

Open Source delivery release for the Nexa Platform REST API.

## Scope

- Java 21 Spring Boot REST API under `/api/v1`.
- Bounded contexts for IAM, Catalog, Sales, Warehouse, Logistics, Invoicing, Promotions, and Shared.
- PostgreSQL profile for Render deployment.
- H2 profile for local and automated tests.
- JWT-based authentication.
- Actuator health check at `/actuator/health`.
- CORS configured for local Angular and Render frontend origins.
- Idempotent `seed` profile for academic demo data.

## Deployment

Render configuration is defined in `render.yaml`:

- Docker web service.
- Managed PostgreSQL database.
- `SPRING_PROFILES_ACTIVE=postgres,seed`.
- `DATABASE_URL` from Render database.
- `JWT_SECRET` as runtime secret.
- `JPA_DDL_AUTO=update`.
- `FRONTEND_ORIGIN` for WebApp CORS.

## Validation

```bash
./mvnw test
./mvnw package
```
