# Database Configuration

Default deployment target is MySQL through the `mysql` profile.

```bash
SPRING_PROFILES_ACTIVE=mysql
DB_HOST=localhost
DB_PORT=3306
DB_NAME=nexa_platform_db
DB_USERNAME=nexa_user
DB_PASSWORD=change-me
```

The schema is generated by JPA mappings during academic verification. Migrations can be introduced later when the deployment environment is fixed.
