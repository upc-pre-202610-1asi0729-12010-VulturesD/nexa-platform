# Project Overview

Nexa Platform is the Java Spring Boot backend for Nexa B2B cold-chain operations.

It exposes RESTful endpoints under `/api/v1` for:

- IAM and JWT authentication,
- catalog products and categories,
- B2B customers and orders,
- inventory and warehouse operations,
- dispatch and delivery tracking,
- invoices, business documents, and payments,
- promotions and buyer portal support data.

The backend is deployed separately from the Angular WebApp and public Website.
